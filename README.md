Foodstitute
=================

Foodstitute is a class exercise for becoming a python developper.
It aims at finding food substitute using the terminal and all of that
amongst 10 food categories which are :

- category 1
- category 2
- category 3
- category 4
- category 5
- category 6

- category 10
